# 쭈불 - 선상 낚시 예약/스케쥴 관리 플랫폼

이 서비스는 원래 외주 개발 의뢰로 시작된 프로젝트였습니다.
특별한 아이디어 하나로 도전한 것이 아니라, 실제 현업 수준의 개발을 경험하고 싶다는 의지에서 출발했습니다.

하지만 개발 외적인 요소에서 발생한 여러 문제로 외주는 무산되었습니다.
다만, 개발 의지만큼은 꺾이지 않았기에 완벽한 프로젝트를 만들어보기 위해 리팩토링을 준비하게 되었습니다.

`demoDay-refactoring`은 resource-server를 clone해 왔습니다.
나의 프로필 보기, 나의 쿠폰 리스트 보기를 제외한 모든 리소스 서버 코드는 제가 작성하였습니다.
- prev (branch) : organization에서 가져온 초기 원본 코드를 보관
- main (barnch) : 리팩토링한 최신 버전 코드

---
## 기술 스택
- **Java 21 / Spring Boot 3.5.6**
- **JPA / Hibernate**
- **MySQL 8.0**
- **Docker / Docker Compose / Nginx**
- **GitHub Actions (CI/CD)**
- **SOLAPI SMS API**
- **AWS EC2 / Blue–Green Deployment**

---
## 주요 기능
### 예약(Reservation)
- 출항 스케줄 관리
- 결제 대기, 결제 완료, 취소 신청, 취소 완료 등 상태 전환
- 예약 시 자동 메시지 발송

### 스케줄(Schedule)
- 한 번에 여러 출항 시간 등록
- 선예약 / 예약 구분하여 등록
- 잔여 좌석 자동 계산 

### 쿠폰(Coupon)
- 관리자 스케줄 취소 시 요일별 쿠폰 발급

### 메시지 자동 전송
- 출항 4일 전 입금 마감 경고 안내
- 출항 3일 전 입금 마감 안내
- 예약 시 자동 메시지 발송
- 관리자 예약 취소 완료 시 자동 메시지 발송
- 출항 확정 / 연기 / 취소 시 자동 메시지 발송
- SOLAPI 연동
